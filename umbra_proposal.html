3. Functional scope — Base Package

This base package includes everything required to launch and operate a professional e-commerce store.

Design & User Experience

  • Custom branded visual design and responsive layouts for mobile, tablet and desktop.
  • Premium micro-interactions and motion on key pages (product detail, checkout) for trust & polish.

Commerce & Operations

  • Product catalogue with categories, variants, search and filters.
  • Cart, secure checkout and payment gateway connectivity (Razorpay/Stripe/PayPal).
  • Order lifecycle management: orders, status updates, invoices, cancellations, refunds.
  • Customer account area with order history and tracking.

Admin & Control

  • Admin console for products, pricing, orders, customers, coupons and basic reports.
  • Role-based access for admins and staff.

Security, Performance & Launch

  • SSL setup and secure transaction handling.
  • Basic performance optimisation and SEO fundamentals (page speed, meta).
  • QA across browsers and devices and production deployment.

Hosting & Domain: Client will handle hosting & domain provisioning (Umbra can assist if required — billed separately).

Timeline: 2 months (subject to timely approvals and provision of content/assets).

Annexure — Technical Appendix (for technical team)

The following details are provided for IT/technical review. These items are implementation options and will be finalised at scoping/technical kick-off.

Architecture & Stack (typical)

  • Frontend: React / Next.js — server-side rendering options for SEO.
  • Motion: GSAP / Framer Motion / Lottie for animations.
  • Backend: Node.js (Express) or .NET Core (API-first).
  • Database: PostgreSQL / MongoDB (based on data shape).
  • Hosting: AWS / Azure / Vercel / Client hosting — CDN + autoscaling recommended.
  • Version control: Git (branching & CI pipelines).
  • Authentication: JWT / OAuth2 for APIs; secure password hashing.

Integrations & Services

  • Payments: Razorpay / Stripe / PayPal (sandbox + production).
  • Emails/SMS: SendGrid / AWS SES / Twilio.
  • Analytics: Google Analytics, Google Search Console, Facebook Pixel.
  • Reporting: Power BI / Looker Studio / CSV / scheduled PDF exports.

Security & Compliance

  • SSL/TLS enforced; data encryption in transit & at rest for sensitive fields.
  • OWASP best practices; input validation and parameterized queries.
  • GDPR-readiness and Indian data protection considerations; data retention policy as required.

Deployment & Operations

  • Minimum recommended server: 2 vCPU / 4GB RAM / 50GB SSD for small-to-medium initial load.
  • Backup schedule: daily DB snapshots, weekly full backups.
  • Monitoring: Uptime monitoring, error logging, performance profiling.

APIs & Extensibility

  • REST APIs for product, order, user, and reporting endpoints.
  • Webhooks for payment and external system events.
  • Clear API documentation will be provided at handover.
